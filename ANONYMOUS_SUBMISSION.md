# Anonymous GitHub Submission Guide

For double-blind peer review, you can create an anonymous version of this repository using Anonymous GitHub.

## Steps to Create Anonymous Link

### Option 1: Using anonymous.4open.science

1. Visit https://anonymous.4open.science/
2. Click "Anonymize a Repository"
3. Enter the repository URL: `https://github.com/1719930244/Awesome-Efficient-AI4SE`
4. Select branch: `main`
5. Click "Anonymize"
6. You will receive an anonymous link like: `https://anonymous.4open.science/r/Awesome-Efficient-AI4SE-XXXX/`

### Option 2: Using anonym.to

1. Visit https://anonym.to/
2. Enter the repository URL: `https://github.com/1719930244/Awesome-Efficient-AI4SE`
3. Click "Create Anonymous Link"
4. Copy the generated anonymous link

## Update Paper with Anonymous Link

Once you have the anonymous link, update the following locations in the paper:

1. **sample.tex line 128**: Replace the GitHub URL in the footnote with the anonymous link

## Important Notes
